# React Basstab
Reactを使ってベースのタブ譜を作れるアプリを開発予定です
## 実装予定機能
- 編集用のモーダル
- ドラッグでのフレット数入力
- ハンマリング、プルオフ、スラーの表示
- 今後の拡張のため入力データをオブジェクトにまとめる
## 開発目的
- Reactの学習
- 苦手なフレーズを繰り返し練習できるようにするため
- 自分だけの好きなフレーズや苦手フレーズを集めたタブ譜集を作るため
- 将来的にRubyを使ってほかのユーザーの譜面を入手できるようにして、弾きたい曲の譜面を見つけやすくしたいため
## 使用技術
- HTML
- CSS
- JavaScript
- React
## データ構造案
```js
column = {
    fret:[1,null,null,null],    //-1(ミュート)と0(開放弦)~21まで
    hammer_on:true,             //ハンマリング
    pull_off:false,             //プルオフ
    bar:false,                  //小節線か否か
}
row = [column,column,...];      //columnの集合で譜面を表現
```
## コンポーネント分割案
#### App
- state管理
- データ保存
- 譜面全体表示
#### Row
- column並べて表示
#### Column
- 1列表示
#### Modal
- columnへの入力受付
- 必要ならUIからデータに変換
